# 陣營

角色的陣營為其大致上的行動方針，ＧＭ認定角色行動違反其行動方針時，可進行強制修正。

    秩序‧善：傾向在規則下行善，並遵守規則。
    秩序‧中庸：聽命行事，不論善惡。
    秩序‧惡：體制下的必要之惡，或以自己的方式守護秩序。
    中立‧善：尊重規則，也不會破壞規則，但自成一派。
    中立‧中庸：守中庸之道，不偏好秩序、混亂、善良、邪惡。
    中立‧惡：只因為一己之利便行惡。
    混沌‧善：以自己認定的方式行善，對規則毫不在乎。
    混沌‧中庸：自由至上，為所欲為的人。
    混沌‧惡：憑本能行惡，唯恐天下不亂。
    混沌‧狂：狂化等級常駐Ｂ（含）以上的從者強制綁定此陣營。一般時不可用。

# 加號與減號

配合 [基本規則－能力的設計](/Basic/Rule.md) 查看

一個從者可獲得５個＋號與１個－號，可以自由分配在素質或固有能力、寶具、戰技、陣地、神殿上。可以選擇不分配。

同一素質、能力最多分配三個＋號。＋號與－號不得混搭。ＥＸ級能力不得配置＋號。

不論能力本身屬於主動或被動，＋號必須在行動時「主動」宣告觸發才會作用，一場戰鬥只能宣告觸發一次。被動技能的宣告固定消耗１ＡＰ且冷卻為一回合，主動技能的宣告則視同發動技能本身，包含原本的效果與宣告的效果。

－號會常駐使素質降低一級並返還１０點額度。

＋號分配在筋力上時，在宣告觸發的行動中可以增加與【＋號數量＊４】同等的傷害骰。－號分配在筋力上時，常駐減少２顆傷害骰（最低至１顆）。

＋號分配在敏捷上時，在宣告觸發的行動中可以增加與【＋號數量＊４】同等的迴避骰、與【＋號數量】同等的移動上限。－號分配在敏捷上時，常駐減少２顆迴避骰（最低至１顆）與１ＡＰ。Ａ－和Ｃ－會常駐減少１格移動上限（最低至１格）。

＋號分配在耐力上時，在宣告觸發的行動中可以增加與【＋號數量＊４】同等的防禦骰與等同【＋號數量＊１０】的傷害閾值。－號分配在耐力上時，常駐減少２顆防禦骰（最低至１顆）與１０點傷害閾值。Ｂ－和Ｄ－會常駐減少１ＡＰ。

＋號分配在魔力上時，每個可以增加２０點ＭＰ上限。－號分配在魔力上時，常駐減少２０ＭＰ上限。

＋號分配在幸運上時，可以選擇其他等同＋號數量的素質或能力，使它們＋號或－號的觸發次數可以增加一次（可重複指定）。－號分配在幸運上時，＋號上限減少一個。Ａ－和Ｃ－會常駐增加１顆大成功門檻。　

＋號分配在能力上時，由這些額度去配置的效果只能在宣告觸發時作用，且只有一回合。觸發能力的＋號時，強制支付等同【加號數量＊５＊能力等階】的ＭＰ。－號分配在能力上時，其「效果額度消耗最少」的效果無法發動（存在複數者則選其中一個），只有宣告觸發的那一回合才具有作用。除非配置於被動，否則一般不必額外支付消耗。

分配在工房、陣地、神殿的場合，每個可以增加２０點效果額度，且這些額度去配置的效果為常駐作用，可與原本的額度混用。

透過＋號、－號觸發的效果不會占用持續時間額度。

在進行百面骰判定時，每個減號使難度＋５，每個加號使難度－５。

# ▶限制與特點

- 不可以直接使用ＡＣＧＮ角色作為從者。
- 召喚出來之前不會知道目前的人理狀況。
- 不開放憑依原作人物。
- 不開放神靈等級的從者（天使或半神人不在此限）。
- 不開放原創從者。
- 不開放以「背叛人理」為前提接受召喚的從者。
- 必須至少擁有一個職階技能、一個固有技能、一個寶具、一個戰技。Caster職階的從者可以不配置戰技改為配置至少一個魔術。Berserker職階的從者可以不配置戰技。

# 從者的素質

從者共有筋力、敏捷、耐力、魔力、幸運五大素質。將依照職階獲得點數以供分配。等級限定為Ｅ～Ａ級，每１０點一級。

筋力：肉體性的力量強度。決定了基本的攻擊骰。

敏捷：速度、反應等複數項目的綜合。決定ＡＰ上限與單次的移動範圍。

耐力：能夠承受的傷害量。

魔力：能夠操作何種程度的魔力。

幸運：運氣的好壞。提供了「骰值重置」的宣言次數。在單一章節內，角色將具有與幸運等級等量的骰值重置次數，章節通關後將重置。每個擲骰判定都可以透過重置宣言重骰最多一次，但一律以新結果計。

可以將能力點數以３０：１０的比例轉換為素質點數，且每轉換２０點素質點數將額外增加１０點消耗。

# 從者的特性與屬性

特性是能用於觸發特攻、特防的各種類別之總稱。

通常而言，經歷過一場戰鬥的場合就會自動公開。

屬性是英靈基本的四大分類，由ＧＭ判定，通常也包含在特性中，不過不同的是，一旦進入靈基一覽就會自動顯示。

「天」乃神靈的降格、顯現，或是真正的神之子嗣。

「地」是出自某地的傳奇人物，或混雜著大量傳說、依憑自然現象而構成的英靈，妖精或魔獸也會被列為此類。

「人」也就是實際存在著、西元以後為人類貢獻的偉人們。

「星」則是人類的希望之象徵，突破亙古難題的獨特存在。

在其中存在「人剋制天，天剋制地，地剋制人，星屬性論外」的法則。

遭遇剋制對象的場合，其藉由普通攻擊、戰技、寶具進行的攻擊骰皆可額外得到1顆骰子的增益。

# 從者分類定義

史實：歷史、傳說、史詩、神話、小說（１９世紀與其後的小說、輕小說皆不包含在內）、知名人物。小說人物必須是地球人（不含作品內的架空宇宙）。

縫合幻靈：藉由將幻靈之間彼此融合，或是將幻靈憑依於英靈之身來達成靈基補完。英靈只能存在一體，最終必須縫合三體。有英靈的場合，可以作為二人一組。

概念：特殊概念的具現體。架空從者、集合體、童話、傳說、學術假說、元素擬人化（如，疾病、影子等）等皆可歸類其中。但必須擁有「大眾廣泛認為的具體形象」的特徵才可以設定。

幻靈：存在事蹟，但仍無法單憑自身成為英靈者，或是靈格不足以成為英靈的人物。

# 職階

目前規則設有七大基本職階與四種Extra職階。基本職階的職階特性為四選二。

職階開放任意選擇，但如果想設定Extra職階，可在跑團確定人選的前一週內向ＧＭ提出申請。

申請的條件是與ＧＭ進行1d100的骰子比大小，五戰三勝制。

毎一團至多開放一人進行Extra職階從者的設定。

各大職階皆具有内建職階能力，可免除點數消耗，若打算選擇者請告知ＧＭ，ＧＭ將會再行調整該角色的能力額度。

配置非職階內建的職階能力時，要額外支付等同等級額度×10點的效果額度，加號每個也需要額外支付10點。

職階技能可以免除封印或無效等效果。

## 職階‧上三騎

### 劍兵（Saber）｜素質點數：１６０

- 聖杯戰爭所能召喚的基礎七種職階之一，擅使刀劍、或以刀劍術聞名之英靈能擔任此職階。
- 職階能力：對魔力　Ｄ、騎乘　Ｅ
- 職階特性：
    - 重擊－劍的普通攻擊攻擊骰數增加１顆。
    - 反制－對近戰攻擊以劍反擊時，其中一次判定骰數增加二顆。
    - 平穩－非重傷狀態且HP高於50%時，防禦行動的骰數增加一顆。以反應進行的場合，於傷害結算後恢復１點HP。
    - 削鐵－以劍對實體物造成傷害時，每次額外造成３點破損度。

### 弓兵（Archer）｜素質點數：１３０

- 聖杯戰爭所能召喚的基礎七種職階之一，具有遠程攻擊能力的英靈能以此職階進行召喚。
- 職階能力：對魔力　Ｅ、單獨行動　Ｅ
- 職階特性：
    - 快攻－戰鬥開始的前三回合，遠程普通攻擊可以將其中一個骰值轉為最大值。
    - 精準－遠程游擊行動的最後一次攻擊傷害不會減半，並且可以觸發大成功。
    - 銳眼－自身視野範圍常駐擴大一圈。
    - 專注－遠程攻擊無視５點傷害閾值，並且可以透過額外消耗1AP使該攻擊的骰數＋２。

### 槍兵（Lancer）｜素質點數：１４０

- 聖杯戰爭所能召喚的基礎七種職階之一，擅使長槍、或以槍術聞名之英靈能以此職階進行召喚。
- 職階能力：對魔力　Ｅ、騎乘　Ｅ
- 職階特性：
    - 追獵－向敵方單位移動時，最大移動格數追加１格。該次移動途中，所接近之目標的藉機攻擊傷害在結算後減半。
    - 識破－反應中的【迴避】行動額外獲得一顆迴避骰，並且迴避後對該目標的首個攻擊骰之骰數增加一顆（無法疊加）。
    - 韌性－受傷時，於傷害計算之後恢復固定２點ＨＰ。
    - 武藝－自身戰技的效果額度額外增加１０點。

## 職階‧下四騎

### 騎士（Rider）｜素質點數：１４０

- 聖杯戰爭所能召喚的基礎七種職階之一，擅長於駕馭騎乘物作戰。
- 職階能力：對魔力　Ｅ、騎乘　Ｅ
- 職階特性：
    - 突破－坐騎移動不會被阻擋中斷，但受阻時最大移動格數－１。
    - 撤離－使用同行時，最大移動格數＋１。
    - 攻城－坐騎移動遭地形、建築阻擋時，將剩餘格數視為等量筋力等級進行破壞行動。
    - 卓越－自身的坐騎額外得到２０點素質點。

### 魔術師（Caster）｜素質點數：１２０

- 聖杯戰爭所能召喚的基礎七種職階之一，事跡中掌握神秘，或是在某項領域有極大成就的英靈，都能以這個職階被召喚。
- 職階能力：道具作成　Ｅ、陣地作成　Ｅ
- 職階特性：
    - 防衛－在陣地進行戰鬥時，自身得到的防禦狀態延長一回合。非主動發起戰鬥時，獲得２點傷害減免。
    - 精熟－每回合指定一個6級（含）以上魔術‧寶具‧魔術類固有技能，其冷卻加速一回合，但該能力在當回合不受其他冷卻加速效果影響。
    - 循環－回合結束時，若該回合中有消耗魔力，固定恢復５ＭＰ。
    - 魔導－得到等同【魔力素質等階＊３０】的魔術點數，只能用於配置魔術。

### 暗殺者（Assassin）｜素質點數：１２０

- 聖杯戰爭所能召喚的基礎七種職階之一，專長於隱密行動與暗殺技巧。
- 職階能力：氣息遮斷　Ｅ
- 職階特性：
    - 先機－戰鬥開始的前三回合，第一次普通攻擊可以將其中一個骰值轉為最大值，骰數增加一顆。
    - 突擊－潛行後，若直到下一回合都未解除，則該回合第一次普通攻擊的大成功門檻減少一顆（無視最低門檻），骰數增加一顆。
    - 暗襲－普通攻擊沒有被察覺、應對時，該次攻擊無視任何傷害減免。
    - 迂迴－若該回合曾受到敵方的直接傷害，下一回合進入隱蔽狀態或發動潛行時固定恢復５點ＨＰ（一回合一次）。

### 狂戰士（Berserker）｜素質點數：１１０

- 聖杯戰爭所能召喚的基礎七種職階之一，擁有「發狂」事蹟的人能以這個職階被召喚。
- 職階能力：狂化　Ｅ
- 職階特性：
    - 嗜戰－受到敵方攻擊或對其造成傷害後，下次普通攻擊的骰數增加１顆。
    - 暴猛－對掩體進行破壞行動時，骰數增加２顆。
    - 無畏－可以無視一次重傷效果（一場戰鬥一次）。
    - 破甲－造成的傷害無視３點傷害減免。

# 武裝化與靈體化

進入特異點前，可自由決定武裝化與否。

戰鬥中的全身靈體化需要消耗整整一回合。

靈體化期間，無法使用任意主動、被動能力，免疫一切物理接觸（仍會受到從者的攻擊或是魔術傷害），魔力傳輸的效率只有一半。

此外，也不計算所有狀態的效果發動與持續時間更新，並且戰鬥行動只能進行移動。

實體化期間，僅有職階技能作用，且會稍微提高劇情間的消耗。

武裝化期間可以正常使用所有能力，但會顯著提高劇情間的消耗。

靈體化狀態下可消耗１ＡＰ來實體化。

除了進入特異點前的武裝化以外，靈體化或實體化狀態下可消耗３ＡＰ與２０ＭＰ來武裝化。

戰鬥前、戰鬥中發動都會計算消耗，Berserker消耗則為兩倍，而御主在場時可以代為承擔一半消耗。這個消耗也可以被【單獨行動】的主動消耗減免影響。

從者之間可以看見靈體。

# 靈核

英靈的核心。具有獨立ＨＰ，固定２０點，歸零時視為死亡。

靈核ＨＰ無法透過任何能力去影響，傷害將會永久持續。每次進入重傷或被暴擊時，靈核ＨＰ－１。

# 從者敵意

因從者之間的戰鬥本能，當相遇時會互相知道對方是從者。

處於氣息遮斷或是無法肉眼察覺、無法被感知的從者無法被辨識。

# 魔力缺乏

從者依靠魔力來戰鬥，雖然魔力歸零時不會消失，但變成負數時就會消失。當魔力歸零時，從者可透過消耗御主的魔力發動技能、寶具、魔術。

# 失去御主

御主陣亡的場合，從者每回合失去４０ＭＰ或於劇情中隨進度減少，低於零之後消失。
