# 跑團原則

    1.「盡可能」不違反世界觀並進行遊戲，後續發現有誤的場合，依據情況決定是否追究。
    2. 玩家選用的形象圖與ＧＭ選用的形象圖（或形象圖對應之實際角色）有重複的場合，由ＧＭ方裁定是否需要更換。
    3. 出現真名相同但設定、解釋上存在衝突的從者時，非ＮＰＣ的從者皆視為來自平行世界。
    4. ＧＭ很討厭ＡＩ繪，所以請盡量不要使用ＡＩ繪當形象。不過，你認為ＧＭ有可能接受的話可以先問問看。
    5. 待補充。

# 懲罰條款

因未先告知的個人因素而引致影響跑團狀況、氣氛、交流時，視同於不尊重其他玩家、ＧＭ投注的心力，將予以一定程度的懲罰。
會被審視的狀況包含但不限於：

    - 未能專注配合跑團事務
    - 在跑團期間分心於外力因素
    - 單團遲到超過10分鐘（以延長後的開始時間為計算基準）
    - 未事先通知即離席
    - 討論途中因個人因素屢次處於狀況外
    - 未關注ＧＭ所發布之公告事項

以上內容，若是ＧＭ未認定達標，但由複數名玩家共同認定的場合，也將列入考量。

若觸犯後有合情合理之緣由，ＧＭ亦會酌情決定是否採計。

## 關於懲罰的內容

    單團內初犯～三犯：每犯一次，則該團內大失敗門檻降低（會低於規則所訂之下限）。
    單團內四犯、屢勸不聽毫無悔改、情況過於嚴重者：下一主線章節的優先度降為「-1」。

若有多次嚴重觸犯，視情況將直接禁止單次參與，或從本團中剔除。

由ＧＭ觸犯的場合，該團相關之全體玩家將得到一次銷過機會。

# 創角限制

玩家加入跑團之後，可以製作一張御主卡與一張從者卡，除此之外有以下數種方式能夠得到新的額度。

    1. 毎使用御主結束一個章節，將追加一張從者卡額度。
    2. 毎使用從者結束一個章節，將追加一張御主卡額度。
    3. 提出自訂活動（詳見下）並舉辦完畢，帶團者將得到一張從者／御主卡額度（擇一）。
    4. 每填寫兩次回饋經ＧＭ認可，將得到一張從者／御主卡額度（擇一），但選擇過其一的場合，需要也選擇過另一邊才能重新選擇。
    5. 對跑團系統之改良做出一定貢獻並受ＧＭ認可，發放內容不定。
    6. 從聖杯戰線賽季獎勵中取得。

# ＧＭ判定補正

御主與從者皆適用，屬於ＧＭ判定的額外加成，根據主觀認定對其人物設定、ＲＰ狀況的滿意程度給予額外的效果額度配點。

此項目由營運群共同判定，一位ＧＭ最多可以給予30點（不會公開各ＧＭ給予的占比以及詳細評定標準，最高60點）。

將於完整參與一次主線或活動章節後配給。

# 曝光度

在特異點探索的時候，可能因為劇情的進行、角色作出特殊的行為或是觸發條件而導致曝光度的上升。

曝光度共有十級，上升可能造成各式各樣的不良影響。

# 根據地

在特異點行動期間，作為休息、整備場所的必要存在，基本上等同於藏身處。

依狀況可能建造複數個，由玩家自行協調。

# 基礎休息行動
通常而言，一次休息會給予兩個行動額度，可能依照狀況增加或減少。

以下休息行動的內容可能依照狀況改變。

    - 休養：每次使用會累積１休養計數。根據人物的【耐力】等階而有使用次數的需求，ＥＸ需要一次，Ａ～Ｂ需要兩次，Ｃ～Ｄ需要三次，Ｅ需要四次，Ｅ－需要五次。戰鬥中再度遭到重傷或暴擊會使休養計數歸零。
    - 完整睡眠：恢復50MP，恢復20HP、全疲勞。占用兩個行動額度。（從者只恢復10MP）
    - 小睡：恢復20MP，恢復10HP、3點疲勞。（從者只恢復5MP）
    - 令咒合成：消耗150MP，恢復1劃令咒（御主限定）。
    - 道具製作：可合計製作Ｎ個等級。必須支付一定程度的魔力。
    - 陣地作成：可以建造一個陣地。必須支付一定程度的魔力。
    - 魔力傳輸：御主將ＭＰ用１：１傳輸給自己的從者。
    - 魔力治療：每消耗10MP可恢復30HP（從者限定）。 
    - 施放技能：發動自己持有的一種技能，人數無限制，但需要根據使用次數增加消耗。
    - 禮裝改修：可以卸下任意數量的制式禮裝能力，並改為一個不同的能力。不過，開頭為「機能」的禮裝能力除了更換的當次休息之外，必須再經過一次休息才能適用。並且「一個章節一次」的效果一旦生效後就無法卸除，直到該章節結束為止。
    - 談話：在夜晚尋找ＮＰＣ交談（同一ＮＰＣ被複數人指定的場合將集體前往）。
    - 修復：無消耗額度。支付ＭＰ修復實體寶具／道具／武器／教會武裝／魔術禮裝／擬似御主的破損度。１點ＭＰ可恢復１點破損。
    - 調查：指定一個已見過的從者，具有【３０％＋幸運等階×５％】的機率得到「詞條」，共進行三次判定。詞條在推斷從者身分的時候可以提供大方向，最多得到十條。每額外累積１疲勞，可以提升每次成功率１５％（最高＋３０％）。此判定由ＧＭ暗骰，成功時得到1條真詞條，失敗時無收穫，骰到90以上未成功的場合得到1條假詞條（大失敗），骰到成功值的1/5（含）以下時得到1個確定真詞條（大成功）。除大成功以外不會知道詞條的真偽。
    - 真名看破：當認為已經針對特定從者掌握足夠線索的時候可以使用。將自己當時持有的關鍵字組合，結合透過劇情觀察得知的線索，並描述排除其他可能人選的過程、如何推導到所認定的解答。經判定成功後可以確定對手的真名、背景資料，並得到對方最主要寶具與最主要固有能力以及弱點資訊（若有）。

# 死亡處理

在特異點探索的時候，角色有可能因為戰鬥或是特殊事件而ＨＰ歸零、死亡，從此無法再度使用。

本遊戲的從者角色視同跟隨前往特異點，而非官方的《Fate/Grand Order》中使用迦勒底設備複製的影從者。

# 戰鬥機制

本遊戲採戰棋式行動，參戰人員的最低需求為一名御主，依照情況可開放總共四～六人參與。

基本處理流程為：準備階段→交戰階段→結算階段。各個角色單獨處理，並周而復始。

## 準備階段

這個階段將會由ＧＭ更新技能冷卻，並以敏捷進行角色行動次序的判定，敏捷相同者擲1d100以大小決定。

部分狀況下亦將考量角色的技能並調整次序。

## 交戰階段

在交戰階段，玩家將依據預先判定的順位依序行動。

玩家的動作額度稱為ＡＰ（Active Point）。

ＡＰ可以不馬上用完，但不可透支，也不會保留到下一輪。

通常只能在此階段使用角色卡上所設定的主動能力。

一旦角色之間產生了接觸、衝突行為，將另外進行雙方的對峙判定。

當ＡＰ使用完畢或是玩家自行宣告結束行動，則由下一位角色行動。

ＡＰ只會在角色下一次行動時回滿。

## 結算階段

這個階段會由ＧＭ計算每回合觸發的效果或是狀態是否結束。

# 反應時間

當兩個不同的單位基於某些原因而產生行動上的衝突，就會進行對峙判定，通稱「反應時間」。

反應時間的觸發方式有許多種，最主要的一種是發起攻擊。

而反應時間能做的事也是各式各樣，以下是確定可用的行動。

    1. 可以施放已經詠唱好的魔術或寶具（不能在反應時進行破棄詠唱）。
    2. 施放能力，包含使用素質、能力的加號或減號。
    3. 施放令咒。
    4. 移動、防禦、迴避、阻擋、保護、潛行、游擊、掩蔽、反擊，以及所有等同這些行動的戰技（沒有標註等同的則不可用）。

不過，在反應時若是使用技能，且超出了對方所使用的ＡＰ量（例如對方進行的是１ＡＰ的攻擊），有可能視為不完全反應。

例：遭受１ＡＰ的攻擊，但自身使用了需要２ＡＰ能力獲得護盾，便會先承受５０％的原始傷害，才會以護盾減免剩餘的５０％。

# 攻擊手法

可以與ＧＭ討論自己角色適合的普通攻擊手法。

同時具有複數種攻擊手法的角色可能會需要支付額外額度。

若是ＧＭ同意，可以在有消耗的前提下配置元素。

# 攻擊屬性

由從者直接引發的能量形式攻擊（包含但不限於火、水、寒氣、風、電、光等）都被視為魔術傷害。御主引發則視情況判斷。

    舉例：
    - 透過斬擊形成風的刀刃不算魔術傷害。
    - 高速摩擦引起的熱不算魔術傷害。
    - 雖然並非自己引發，但類似於「向上天祈求」而發生的能量攻擊依然視為魔術傷害。


由從者造成的非物理異常狀態（例如概念層面的影響）都被視為魔術性效果。

原理上與「命運力」（即幸運）有關聯的效果（包含但不限於因果律能力）不在此限。

# 神秘

更純粹的神祕會無效化稀薄的神祕。故即使是能行使神秘的魔術師，也無法與作為強大神秘化身的從者直接對抗。

遊戲中，御主對峙從者或持有神秘的單位的時候，自身的行動會受到以下影響：

    1. 不帶魔力的物理攻撃無法造成傷害，且純物理手段引起的負面狀態（壓制除外）一律無效化。
    2. 非寶具或魔術禮裝的軍火類攻撃無法造成傷害。
    3. 有效攻撃的傷害下降５０點，狀態則下降２０點（包含提升持續傷害的元素，但最低會剩下１點）。
    4. 技能本身有設定資料配合者，6級（含）以下能力賦予的魔術性負面狀態無效化；若在7級～8級，則改將該狀態視為降低一階（最低Ｅ級）；9級不受影響。即便配置足夠等級，ＧＭ主觀判定程度與等級不符的魔術，同樣以無效化處置。

# 從者優勢

從者與一般生物在肉體的能力上具有一定程度的分別，為體現該區別，從者將會受到以下效果強化：

    1. 傷害骰＋3d6。
    2. 防禦骰＋3d6。
    3. 迴避骰＋3d4。
    4. 最大ＭＰ＋３０。
    5. 移動上限＋１。
    6. ＡＰ＋２。
    7. 被藉機攻擊時，中斷詠唱的傷害門檻增加１０點。
    8. 戰鬥時的行動優先度＋１，並且一定高於同等級的御主。

# 重傷與暴擊

耐力素質會決定一個單位的傷害閾值，意即對傷害耐受感到疲勞的數值門檻。

單次承受的傷害突破傷害閾值時，觸發重傷。

重傷有部位區分，雙手、雙腳、頭部、身軀，一共六個部位。部位一旦進入重傷就是永久存在。

重傷只能透過「休養」與能夠直接解除重傷的能力來解除。

觸發重傷時，會依照狀況決定直接選擇或是隨機抽選以下其中一種重傷影響，再追加一個固定的部位影響。

若骰中斷肢，將額外進行耐力判定，有【100%－耐力×10%】的機率觸發，未觸發則改為其他重傷之一。

耐力判定將由遭受重傷的單位進行擲骰。

斷肢不會累計後遺症計數。

於御主身上造成的場合，斷肢無法被治癒。

## 重傷影響列表

    - 骨折：該部位無法動作，但沒有脫離身軀。頭部與身軀不會骨折。
    - 斷肢：該部位無法使用，脫離身軀，且只能用強效的治療手段才能恢復。頭部與身軀不會斷肢。
    - 脫力：該部位造成的傷害下降一半。身軀不會脫力。
    - 碎骨：該部位每次動作都會失去５ＨＰ。身軀則變為受撃時失去５ＨＰ。
    - 出血：每回合失去５ＨＰ，且受到的治療效益減半。
    - 昏厥：無法行動，且失去意識。

## 部位影響

    - 昏眩：頭部限定。指向性行動有５０％機率轉為其他方向。
    - 內傷：身軀限定。每回合失去１０ＨＰ，且該部位每次受撃時失去５ＨＰ。
    - 無力：雙手限定。造成的傷害減半。
    - 瘸腿：雙腳限定。移動最大範圍減半。

重傷期間，再度受到能突破傷害閾值的傷害時，該次傷害觸發暴撃。暴撃的傷害固定增加５０點，最終傷害不會低於５０點，並無視於防禦相關的戰鬥行動與強化效果。

## 御主獨立內容

主線章節期間，若御主的重傷沒有在章節期間治癒，則會累積３點後遺症計數。
此外，並非以能夠直接解除重傷的能力去解除重傷的話，會累積１點後遺症計數。

後遺症計數：
    到達以下點數後，會產生各種不良效果（可疊加）。
    ０～４：暫時無大礙。
    ５：傷害閾值永久下降５點。
    １０：ＨＰ上限永久下降１５點。
    １５：被觸發重傷時，同時也會觸發暴擊。
    ２０：疲勞上限永久下降２點。
    ２５：下一次承受的斷肢以外重傷無法被治癒。

# 強敵

在遊戲進行時，有時候會出現具有大量ＨＰ的敵人，ＨＰ遠超過包含玩家角色在內的一般單位，甚至具備特殊效果。

這類敵人通常是從者、ＢＯＳＳ從者、特別強大的怪物或魔獸。

更困難的狀況下，可能會出現具有複數ＨＰ的敵人，無法透過一次性的撃倒來完全攻略。同時，當該種敵人的其中一條ＨＰ歸零時，亦有可能使用特殊技能。

撃破其中一條ＨＰ之後若撤退，該ＨＰ並不會於短時間內恢復，但長時不繼續攻略即有可能再度恢復。

# 行動額度

行動額度，通稱ＡＰ，是進行各式戰鬥行動的籌碼，象徵著在同一段時間內能進行多少動作。

每個單位基礎具有３點ＡＰ，根據敏捷可以獲得額外額度。

耐力：１／２／２／３／３／５（隨等級變動）。

等級為Ｄ－與Ｂ－的場合會減少１點。

敏捷：每等級＋１，ＥＸ視為８等級。

# 角色數值基準

ＭＰ上限：７５＋２０／４０／６０／８５／１１５／２５０（隨魔力等級變動）。

ＨＰ上限：９０＋１５／３０／４５／６５／８５／１８０（隨耐力等級變動）。

傷害閾值：５０＋５／１０／１５／２５／３５／８０（隨耐力等級變動）。

ＨＰ低於一半的時候，傷害閾值減少１０點。

ＨＰ低於１０％的時候，傷害閾值減少２０點。

# 詠唱與施放

大多數的主動寶具與魔術都具有詠唱要求，且以用於詠唱的ＡＰ量作為計量單位，只要詠唱到需求量即可滿足發動條件。

一回合內可以消耗任意數量的ＡＰ進行詠唱，但不同招式可能存在不同的最低需求量。

（範例：該魔術需要合計９ＡＰ的詠唱，且一次詠唱至少需要３ＡＰ。）

已經在詠唱一個能力的狀況下，不得進行第二個能力的詠唱。

默認狀況下，完整的一個回合沒有繼續詠唱就會自動中斷，若詠唱完畢的能力在下一回合結束前沒有施放，同樣視為自動中斷。

詠唱期間無須支付該能力的消耗，直到施放時才行支付。

能力遭到封印或是冷卻期間無法詠唱。

詠唱依照情況可能會因為受到外力干涉而中斷，詳見下方《[打斷詠唱](#打斷詠唱)》。

設定參考：

    一工程的魔術需要1AP詠唱時間。
    一小節的魔術需要3AP詠唱時間。

# 對能力發動的察覺

他人進行詠唱時，若詠唱者位於自身的聽覺偵查範圍內將自動得知。若處於隱蔽狀態則該隱蔽也會解除。

    １級～４級的直接攻擊型‧直接防禦型‧負面效果型能力在詠唱時，最後的３點ＡＰ會於詠唱者身邊產生魔力的反應。

    以詠唱者為中心，5×5範圍內具有至少Ｅ級魔力感知能力的單位可以自動得知，9×9範圍內必須具有至少Ｄ級的魔力感知能力才能發現。

    ５～６級的直接攻擊型‧直接防禦型‧負面效果型能力在詠唱時，最後的３點ＡＰ會對全場都產生魔力的反應。至少擁有Ｅ級魔力感知的單位皆自動得知。

    ７級（含）以上的範圍攻擊型‧負面效果型能力或５級（含）以上的寶具在詠唱時，將在詠唱期間對全場都產生魔力的反應。至少擁有Ｅ級魔力感知的單位皆自動得知。

這類在全場產生的魔力反應，默認無法明確掌握到詠唱者的詳細位置，但可以視為衝突判定的發動前提。

解放寶具必須以正常以上的音量喊出寶具的真名，故所有單位都能自動察覺在聽覺偵察範圍內的寶具解放。

場上具有至少Ｅ級魔力感知能力的單位都能夠察覺到４級（含）以上且不帶有【隱密】元素之能力的解放。

# 基本與效果偵察

基本偵察部分，視覺、聽覺為以自身為中心的5x5，嗅覺為以自身為中心的3x3。

超出視覺與聽覺的偵察範圍兩圈以內的範圍屬於「偵察模糊範圍」，雖然能大致上知道該區域發生的事（比方說劇烈的震動、爆炸等聲光反應），但無法對其中的單位做到明確分辨（故無法指定其作為目標）。

往「偵察模糊範圍」發動攻擊時，若不具有必中效果，有可能會產生目標或範圍的偏移因而扣減傷害（由ＧＭ認定），且目標的迴避減傷將可以達到１００％。在此前提下，攻擊者持有必中效果而目標持有庇護效果的場合，迴避將保持５０％減傷不變。

當有複數偵察效果為相同的感覺類型，並且同時存在時，其所賦予的偵察效果將會互相疊加。

透過疊加得到的額外範圍最多５圈。

此外，【教會勢力】‧【魔術師】身分的御主以及從者，自動依據其魔力素質等級得到一定範圍的被動感知效果。

    Ｅ／無。

    Ｄ／3×3。

    Ｃ／5×5。

    Ｂ／7×7。

    Ａ／9×9。

    ＥＸ／15×15，且感知被視為Ｃ級。

這項感知效果視同針對魔力反應的Ｅ級感知。任何單位以不帶有【隱密】元素的能力支付魔力時，能感應到魔力的反應，但無法分辨來源是敵是友。

# 基本與進階行動

戰鬥行動分為三大基本類型：攻撃、防禦、移動，各自消耗１ＡＰ。此外，存在特殊的「觀望」、「警戒」等。

戰鬥行動可以透過三種方式獲得追加效果，分別是強化、複合、連攜。

強化的額外消耗為１ＡＰ，每支付１ＡＰ在同種類行動上時，使該行動升級。

複合的額外消耗為１ＡＰ，視同將第二種類型的基本行動融合。

連攜則是在友軍在一定距離內行動時可以自主追加，與其使用相同的基本行動，無額外ＡＰ消耗。

## 基本行動Lv.1

以下將列出基本戰鬥行動的使用效果。

### 攻撃

- 建立在肉搏、兵器上的近戰攻撃手段。
- 詳細手段需要填在角卡的「攻擊手法」欄位，符合的範圍由ＧＭ決定。
- 默認狀況下，一次行動只能對攻撃範圍內的單位造成一次物理傷害。
- 基礎傷害以筋力作判定，等同【（１＋筋力等級）d6】點。

### 防禦

- 呈現守備姿勢承受攻撃。
- 默認狀況下以耐力作判定，等同【（１＋耐力等級）d6】點。
- 對範圍攻擊最多減免該傷害的一半。

### 移動

- 徒步往指定方向移動。
- 默認狀況下，根據敏捷決定最大移動格數，Ｅ～Ｄ級１格，Ｃ～Ｂ級２格，Ａ級３格，ＥＸ級７格。
- 不論如何提升，最大移動格數都具有上限，最多提升到【基礎值＋敏捷等級】（加減號、從者優勢不受此限）。
- 無法越過敵人或是大小大於自己的單位。

### 觀望

- 觀察戰場態勢，等候更好的時機才進行行動。
- 必須在自己的回合一開始的時候就決定是否進行，若已執行其他行動或施放任何能力則不可進行。
- 將自己在該回合或下一回合的行動次序降低，每降低1點需要支付1AP。
- 行動次序不會自然復原，但可以事先宣告並於下回合完全復原。
- 復原之後，在回合開始時，會根據恢復的程度每1點減少1AP。
- 降低後的行動次序必然比同次序的其他角色低，若是同樣發起觀望的角色則以發動的次序決定。
- 這個行動沒有進階型。

### 警戒

- 對自己的周遭或是敵方的動靜提高警覺。
- 這個行動的消耗為３ＡＰ，且必須是自己回合的最後一個動作。
- 使用後進入警戒狀態，持續到自己的下一回合開始。
- 警戒期間適用以下效果：

    - 在偵察或感知範圍內若有任何敵人移動，自身都可以追加發起移動或其進階行動。
    - 可以宣言單一目標的任意一種行動，符合條件時，自身可以對其進行反應，不過反應時只能使用部分行動。
    - 可以正常應對潛行單位行動以及受到反擊而誤擊自身的攻擊。
    - 迴避骰增加１顆。
    - 被無法應對的攻擊命中時，該次傷害的骰值翻倍（即骰出的數字翻倍，加成保持原本）。
    - 一旦有敵方單位進行詠唱且能遭到偵察或感知，則在其詠唱到解放期間的所有行動時，自身都可以追加發起移動或其進階行動，以及透過警戒追加反應時所能發動的行動。

- 警戒的效果不會作用於自身無法察覺到的隱匿中目標，亦不作用於隱匿解除後的首個行動。
- 這個行動沒有進階型。
- 警戒可以宣言的行動：攻擊／強攻／破壞／移動／阻擋／發動技能／發動寶具。
- 透過警戒追加反應時可以進行的行動：防禦／迴避／保護／發動技能／發動寶具。

## 基本行動Lv.2

以下將列出額外投資１ＡＰ的進階行動使用效果。

### 強攻

- 「攻撃」行動的進階型，帶有該行動的基本效果。
- 對「防禦」行動具有剋制效果，可以無視於防禦骰的減免。
- 累積的破損度增加１００％（不會作用於視同強攻的戰技上）。

### 阻擋

- 「防禦」行動的進階型，帶有該行動的基本效果。
- 對「移動」行動具有剋制效果，在自身移動範圍內的單位移動時，移至該路線上並阻斷移動。
- 被阻擋後，直到回合結束為止，對象無法再往阻擋者的方向移動。
- 可同時指定一個3×3內的相鄰單位，由自身代替其承受單一方向的一半傷害，但無法代為承受貫通、範圍攻擊的傷害。
- 無法阻擋比自身巨大的單位。
- 這個行動可以在單位發生了擊退以外的移動或是相鄰單位遭受反擊以外的攻擊時追加發動。

### 迴避

- 「移動」行動的進階型，帶有該行動的基本效果。
- 默認狀況下以敏捷作判定，等同【（１＋敏捷等級）d4】點，若與攻擊者相隔一格（含）以上，每格增加1d4點。
- 對「攻撃」行動具有剋制效果，不論如何都能至少降低一半傷害，面對範圍傷害時只要能移出攻擊範圍，也能作用。
- 如果骰出的數值將攻擊傷害抵銷，則不會被賦予該次攻擊所帶來的特殊狀態。
- 迴避時的移動格數最多２格，敏捷特質為Ａ級的場合最多３格。

## 基本行動Lv.3
以下將列出額外投資２ＡＰ的進階行動使用效果。

### 破壞

- 「強攻」行動的進階型，帶有該行動的基本效果，並且也無視「保護」的防禦骰減免。
- 可損毀筋力等級（含）以下的地形或建築（無標註視為Ｅ級）。
- 對「掩蔽」行動具有剋制效果，撃破對方的掩體時不會減少造成的傷害。
- 累積的破損度增加１００％（不會作用於視同破壞的戰技上）。

### 保護

- 「阻擋」行動的進階型，帶有該行動的基本效果。
- 指定一個3×3內的相鄰單位作為保護對象，由自身代替其承受單一方向的傷害，貫通、範圍攻撃則代為承受一半傷害。
- 對「反撃」行動具有剋制效果，該次反撃轉由自身承受，受保護的對象可照常造成傷害。
- 相鄰單位受到「攻擊」、「強攻」、「反擊」、藉機攻擊時可以追加發動。

### 潛行

- 「迴避」行動的進階型，帶有該行動的基本效果。
- 若敵方已指定其他單位為目標並行動，則無法應對正在潛行的單位與其結束後的首個ＡＰ。
- 對「游撃」行動具有剋制效果，使游撃途中造成的每次傷害各自減免一半。
- 不受迴避的移動格數限制影響。
- 大小不為1×1的單位無法使用。
- 如果在潛行期間自曝位置（如解放寶具、造成聲光效果），則仍舊可以被應對。

## 複合行動

以下將列出複合行動的使用效果。

### 游擊

- 在行動的同時發起多次的快速攻撃。
- 「攻撃」行動與「移動」行動的複合型，具有各自的部分特性。
- 可移動格數的計算方式與「移動」行動相同，但每次移動一格後可發動一次普通攻撃。
- 此行動發動的每次普通攻撃傷害各自只有正常「攻撃」行動傷害的一半。
- 不一定每次移動都必須進行攻撃，但過程中必須至少攻擊一次。
- 大小不為1×1的單位無法使用。
- 使用＋號增加的攻擊骰數只能在任意的其中一次擲骰適用。
- 反應時，一次行動即可應對所有游擊攻擊。

關於游擊，有限定部分武器才可進行。詳情請見【[補充資料篇](/Extra%20Info/Extra%20Info.md)】的【攻擊手法】。

### 掩蔽

- 快速尋找地形物並躲至其後方抵禦襲撃。
- 「防禦」行動與「移動」行動的複合型，具有各自的部分特性。
- 可將地形物作為掩體阻擋攻撃，只要掩體沒有破壞，自身就不會受到傷害。
- 若掩體被破壞，則該次攻撃傷害減半，且會再度計算與「防禦」行動同等的傷害減免。
- 部分狀況下無法成功減傷，例如掩體阻擋方向錯誤、掩體無法完全遮蔽等。

### 反擊

- 針對敵方的攻撃做出相應行為反制。
- 「攻撃」行動與「防禦」行動的複合型，具有各自的部分特性。
- 以最大反撃次數為限，對每次受到的攻撃進行一次反撃判定，而且即使遭到圍攻也可以應對至多三個單位的攻擊。
- 最大反撃次數根據敏捷作判定，次數為【２＋敏捷等級】。次數用盡後，每次反擊需要消耗１ＡＰ。
- 如果透過反擊將攻擊傷害完全抵銷時，則不會被賦予該次攻擊所帶來的特殊狀態。
- 使用＋號增加的攻擊骰數只能在任意的其中一次擲骰適用。

## 連攜行動

以下將列出連攜行動的使用效果。

### 圍攻

- 與他人共同對單個目標發起攻勢。
- 至少兩個單位對同一單位發起「攻撃」行動時觸發。
- 默認狀況下，被攻撃者只能對一個單位的攻撃進行應對。

### 聯合

- 與他人共同對襲來的攻撃進行防禦。
- 至少兩個單位在相鄰的狀況下，對單一攻撃發起「防禦」行動時觸發。
- 可以將全體使用者的骰值加總並減免該次攻撃傷害，共同承受減免後的傷害。
- 針對大範圍攻擊時無效，僅限近戰、遠程、貫通等。

### 同行

- 將身邊的同伴帶走並且共同移動。
- 至少兩個單位在九宮格內的狀況下，向同一個方向發起「移動」行動時觸發。
- 默認狀況下，以發起同行者決定移動格數上限。
- 攜帶一人同行時的移動上限減３（最低至１），二人（含）以上的移動上限則是減３或除２（取較低，最低至１，無條件捨去）。
- 原則上以雙手各拉一人為限。
- 無法攜帶大小大於自己的單位。
- 若要使用【穿行】或【閃現】元素的效果，則按照原理可能無法攜帶不具這些效果的單位，即使可以攜帶，那些單位也不會受惠於這些元素的影響。

# 藉機攻擊

如果交戰時，敵人處於自身近戰、遠程的攻擊範圍時又滿足某些特定條件，則有可能進行一次不消耗ＡＰ的攻擊行動。

    1. 進行了【移動】、【迴避】、【掩蔽】、【游擊】等行動。
    2. 空過回合。
    3. 從戰場撤退。
    4. 詠唱。
    5. 使用道具。
    6. 使用令咒。

藉機攻擊會受到【迴避】、【掩蔽】的減傷影響。

游擊的每一次移動都可以觸發【藉機攻擊】或【反擊】。

飛行單位與地面單位的行動不會對彼此觸發近戰的藉機攻擊。

以遠程的形式發動【藉機攻擊】時，除非對方在二格以內，否則每次將消耗１ＡＰ進行。此外，對超出有效射程的單位，無論如何都無法發動藉機攻擊。

無法使用額外追加ＡＰ的方式進行藉機攻擊。

# 衝突判定

原理允許的場合，若持有直接傷害型的主動能力，[若察覺到敵人正在詠唱攻擊型的能力（透過已經掌握的情報、魔力反應、ＮＰＣ的告知等）](#對能力發動的察覺)，也可以發動或詠唱自己的傷害型能力進行對抗。

可能不被允許進行衝突判定的場合包含：角度被固定的攻擊（必須由上而下等）、沒有實際的形體（即指定對象發動）等。

其中的某些案例可能會被允許「與之進行對抗」，但不被允許「用於對抗」。

這個場合下，除了參與衝突判定的角色外，直到衝突判定結束，其他單位無法再進行干涉。

若是當前可以用於詠唱的ＡＰ不足，則判定結束後直接視為ＡＰ歸零。

追加如果他人在周邊3×3的位置詠唱‧發動近戰攻擊型能力，則遠程‧範圍型寶具在詠唱時的ＡＰ消耗視為增加３點（僅用於判定，與實際詠唱消耗無關），即便成功擊發，在進行「範圍對單體」的提升判定時，也僅會增加１００％威力。

追加詠唱進行對抗的場合，若詠唱需求較對手長，則無法成功對抗（視為無法反應）；否則，即便詠唱需求較對手短，也視為必定與對方的攻擊同時發出。

不論攻擊者的詠唱需求為何，都可以進行[破棄詠唱](#破棄詠唱)（不得在得知詠唱需求不足後更改）來進行應對。

得到攻擊者允許的場合，即便對抗者的詠唱需求較長，也可以同時擊發。有時會考量角色的陣營與當下情況決定強制允許（例如秩序‧善的單位進行公平對決等）。

觸發衝突判定時，若其中一方的傷害高出另一方５０％（餘數無條件捨去），則可將攻擊壓過，對敵人造成差值之傷害（被壓過的一方無法反應）；否則，將在攻擊的接觸點引發5×5～11×11不等的爆發，傷害等同雙方攻擊的差值。

只要攻擊者或對抗者任何一方使用的傷害能力為近戰攻擊，將不會產生爆發，且近戰攻擊無法壓過遠程攻擊。

若其中一方為範圍‧多格寬遠程攻擊，而另一方為近戰‧單格寬遠程攻擊，僅在進行高低判定時，範圍‧多格寬遠程攻擊的傷害值提升２００％（但計算傷害差值時，依舊採用原本傷害，最低不會低於原傷害的一半）。

一般攻擊無法用於觸發衝突判定。

# 羈絆等級

在特異點探索結束後，或多或少都會更加適應同伴的行事風格，使得彼此之間的默契強化。

每次結團時，所有參與的角色與自己的契約對象將會提升一級羈絆，起始為Lv0，最大為Lv5。

此提升為默認，玩家可根據劇情狀況提出不提升的申請，若實際狀況極差，也可能由ＧＭ直接判定不予提升。

除了跑團以外，可以在休團期間自行提出進行「幕間物語」，進行角色之間的日常互動，將視情況提升羈絆等級。

Lv1／以契約對象為目標的主動單體能力給予的ＨＰ治療量上升１５點、ＭＰ恢復量上升１０點、護盾量上升１０點、攻擊傷害影響量上升５點、防禦力影響量上升３點。

（以上效果最多持續一回合，持續治療或恢復效果只有第一回合觸發，超量只有一次觸發，只要符合對應效果就會追加。）

Lv2／解鎖[羈絆行動](#羈絆行動)。

Lv3／無須消耗等級額度獲得最高６級的「羈絆技能」，其效果額度為關聯之能力效果額度（以減免後計算）的平均值。羈絆技能不會影響等級額外補正。詳見[補充資料－羈絆技能](/Extra%20Info/Extra%20Info.md)

Lv4／雙方可自選一個能力，無點數消耗提升等級。

    1～2級：提升3級，額外得到50點效果額度。
    3～5級：提升2級，額外得到40點效果額度。
    6～7級：提升1級，額外得到30點效果額度。
    8級以上：不提升。

Lv5／羈絆技能等級上限提升至９級，並額外得到５０點額度（依然不可超出等級所允許的效果量上限）。

羈絆到達Lv5後，必須再進行一次幕間物語才能解鎖效果。

等級５之後，必須使用【旅路的證明】突破羈絆上限，才能解放等級６～等級１０。羈絆上限的突破可以由組別中的任何一方消耗道具進行。解放後，便能透過進行主線劇情來增加羈絆等級，每次進行活動或幕間則會累積１進度點，滿２進度點後以升級計。

玩家每完成三個章節或定期活動，便能得到一個【旅路的證明】（同一活動只取一位角色參與即可）。

Lv6／令咒勵起Ⅰ：御主發動令咒的該回合內，從者的筋力、敏捷、耐力上升一級。

Lv7／靈基再臨：從者得到6級等級額度與140點效果額度（只能用於強化或追加固有技能或寶具）。

Lv8／令咒勵起Ⅱ：御主發動令咒的該回合內，從者攻擊、防禦、迴避骰數+3（各一次）。

Lv9／令咒勵起Ⅲ：御主發動令咒的該回合內，從者的寶具傷害值上升20點（一次）。從者沒有傷害型寶具的場合，【令咒勵起Lv.2】提供的骰數再+3。

Lv10／靈基再臨：從者得到6級等級額度與140點效果額度（只能用於強化或追加固有技能或寶具）。

每次靈基再臨，主從必須與ＧＭ約時間進行再臨戰鬥，對決和從者具有某種緣分關聯的存在。

該戰鬥即便戰死、敗北也不會對一般遊戲進行的角色卡有所影響，但無法使用【令咒】、【強制解放】、【幻想崩壞】。

羈絆Lv.10的場合，除靈基再臨之外，必須進行一次「幕間物語」才作用。

## 羈絆行動
以下是在到達一定的羈絆等級時才能解鎖的特殊行動。

### 指揮

- 向對方進行戰術上的指揮。
- 消耗１ＡＰ之後，可以指定將自身的１～３ＡＰ交予對方行動。
- 即使他人受到指揮，也不會影響之後的行動次序。
- 一回合最多一次。

### 圍攻（絆）

- 等同上位版的「[圍攻](#圍攻)」行動，具有該行動的基本效果。
- 契約對象加入時，該行動期間雙方造成的傷害量上升５點。

### 聯合（絆）

- 等同上位版的「[聯合](#聯合)」行動，具有該行動的基本效果。
- 契約對象加入時，該行動期間雙方額外獲得１０點護盾值。

### 同行（絆）

- 等同上位版的「[同行](#同行)」行動，具有該行動的基本效果。
- 契約對象加入時，該行動期間移動上限上升一格（不會遭到同行本身的減免影響）。
- 契約對象加入時，該行動期間自身帶走的契約對象獲得２０點護盾值。

# 額度的使用

角色持有的能力、素質、能力的效果、回合數都有額度的限制，且彼此是不同的事物。

回合額度為一個能力總和計算，也就是所有效果（含立即發動，計為一回合）合計的回合數。

配置元素時會按照額度欄的數字增減效果額度。包含百分比的時候會直接變動原始的最大值。定值的時候若為負數將額外獲得額度，正數則扣減額度。

# 能力的設計

配合 [御主規則－加號與減號](/Master/Master.md)、[從者規則－加號與減號](/Servant/Servant.md) 查看

角色的能力在一開始不具備任何功用，必須配置效果表上的項目才能夠產生實際作用。御主的制式禮裝與從者的職階技能除外，不需要配置元素而直接取用表格提供的效果。

另外，從者的職階技能將以等級計算消耗。

    Ｅ級／2等級。

    Ｄ級／3等級。

    Ｃ級／4等級。

    Ｂ級／5等級。

    Ａ級／6等級。

    ＥＸ級／9等級。

御主的特質則會消耗與階級同等的等級額度。

有一些效果是必須分配的，有一些則不是，此外若無特別註明，效果無法點選兩個名稱與等級皆相同的，必須注意。一項能力最多能分配９個效果。

能力將具有「等級欄位」、「效果量欄位」、「階級欄位」，等級與效果量必須個別配置數字，階級則只是帳面設置，並用於配置＋號、－號，且ＧＭ可能視情況調整。配置＋號會返還50點效果額度。配置－號會返還3個等級額度。

等級欄位：不再計算Ｅ～ＥＸ的「階級」，而是改在能力上配置１～９的「等級」，用以決定「最多能配置多少的效果」。

    等級１／最多８０點。

    等級２／最多１００點。

    等級３／最多１２０點。

    等級４／最多１４０點。

    等級５／最多１８０點。

    等級６／最多２１０點。

    等級７／最多２４０點。

    等級８／最多２９０點。

    等級９／最多３５０點。


請注意，配置的效果量將以「受到減免（如制約、返點）之前」的數字計算。

效果量欄位：在「等級」所提供的範圍內，任意決定自己要將多少的「效果額度」配置在該技能上。

除了效果額度以外，能力也會根據等級獲得不同的持續時間額度。

能力等級所能獲得的持續回合額度：

    等級１／２回合。

    等級２～３／３回合。

    等級４～５／４回合。

    等級６～７／５回合。

    等級８／６回合。

    等級９／７回合。

此外，效果兌換成等級為20:1，等級兌換成效果為1:10。請在欄位上以正數、負數表示。

效果可以自由選擇立即發動或是消耗額度設定持續回合。立即發動（傷害、治療、恢復等等）視為一回合。

如果合情合理，可以向ＧＭ申請隱藏自身部分能力的資訊，也可以選擇單純的不告知隊友，但不告知的狀況是可以被隊友的能力或是迦勒底的檢測系統看破的。

可以提出將特定效果視為「副作用」，ＧＭ會依照情況決定該副作用減免的效果額度量。副作用減免的效果額度量不會超過最大值的２５％。

從者的職階技能與御主的特質也屬於能力的一部份，但它們不能配置額度，同時，固定為被動能力，效果亦為固定。

ＧＭ有權依照原理對效果追加任何發動條件，此狀況下會依照情形追加效果額度量，若元素已達９種，必須選擇一個刪除並替換為【制約】效果。

冷卻時間與消耗由ＧＭ依照原理與平衡需求填寫，並且回合數至少會與能力效果持續時間最長者同等。冷卻統一從施放完的下回合開始計算。

另，在元素一覽中以「藍色」標記的元素不占用回合額度，以「紅色」標記的元素會占用回合額度，以「黃色」標記的元素不占用回合額度也不會因為被動而翻倍。

## 等級重複補正

為避免有過多且過強的能力存在而設置。

    ５～６級能力數量超過２個時，每超出一個將額外減少1點等級額度。

    ７級能力數量超過２個時，每超出一個將額外減少2點等級額度。

    ８～９級能力數量超過１個時，每超出一個將額外減少3點等級額度。

## 元素限制與補正

若是在等級過低的技能上配置高級元素，將增加消耗。

    ＥＸ級元素：要求至少８級，否則每項額外+30。

    Ａ級元素：要求至少６級，否則每項額外+10。

    Ｂ級元素：要求至少５級，否則每項額外+10。

而如果１級～５級能力只配置Ｅ級～Ｃ級、I級、無後綴、非傷害系、非防禦系之元素時，可得到額外效果額度。

    １級～２級：３０。

    ３級～４級：２０。

    ５級：１０。

不過，在藉由元素補正得到效果額度的場合，必須要至少配置10點。

從者戰技所配置的持續性效果數量在兩項（含）以內時，無法適用元素補正。

    －５：御主將魔力給予從者的效益減半（包含日常行動與戰鬥中使用御主魔力等）。

    －４：契約從者需要自身支付一劃令咒才能解放寶具。

    －３：契約從者所有魔力消耗＋１０。

    －２：契約從者最大ＨＰ－１０，最大ＭＰ－１０。

    －１：契約從者造成的攻擊傷害－５，受到的傷害＋５。

    ０：無變動。

    ＋１：契約從者造成的攻擊傷害＋５，受到的傷害－５。

    ＋２：契約從者最大ＨＰ＋１５，最大ＭＰ＋１５。

    ＋３：契約從者所有魔力消耗－１０（不可低於原本的一半）。

    ＋４：寶具造成的傷害×２（一場戰鬥一次）。

    ＋５：御主將魔力給予從者的效益翻倍（包含日常行動與戰鬥中使用御主魔力等）。

# 破棄詠唱

發動魔術或寶具時，可以自行決定追加魔力消耗，免去詠唱限制。

不過，藉由破棄詠唱發動的能力，若有傷害則造成的傷害減半，否則失去其額度消耗最多的一項效果（有額度重複的場合，以ＧＭ判定）。

    需求為１～５ＡＰ：每ＡＰ＋１０點ＭＰ。
    需求為６～１１ＡＰ：ＭＰ消耗×２。
    需求為１２～１４ＡＰ：ＭＰ消耗×３。
    需求為１５ＡＰ以上：ＭＰ消耗×４。

# 強制解放

從者與御主在發動魔術、魔術禮裝、魔術類寶具、具有實體器物的主動寶具等能力時，若處於冷卻期間，可以藉由支付HP，強制無視冷卻發動。

由從者執行的時候，只要是寶具就能使用強制解放。所需要支付的HP只有一半，但必須根據免去的冷卻回合數每回合累積靈核傷害。

以此方式支付的HP，不會觸發「失去HP」、「受傷」等制約條件的判定。

    １～３級能力／支付20HP。

    ４～６級能力／支付40HP。

    ７～９級能力／支付60HP。

此外，使用【強制解放】的時候，依照魔力素質的等級，可以有數次機會免去其代價（包含靈核傷勢）。

    Ｅ級、Ｄ級／０次

    Ｃ級／１次

    Ｂ級／２次

    Ａ級／３次

    ＥＸ級／５次

這個次數將在章節結束後重置。

# 打斷詠唱

除了條件不足以致無法施放外，期間因外力或特殊原因而滿足以下任一條件則詠唱將被強制打斷：

    1. 遭到【[強攻](#強攻)】並被一次性打出４０點（含）以上的有效傷害。
    2. 遭到【[藉機攻擊](#藉機攻擊)】並被一次性打出２５點（含）以上的有效傷害。
    3. 一次性受到８０點（含）以上的有效傷害。
    4. 被賦予會影響行動的不良狀態並生效。
    5. 一次性被擊退兩格（含）以上。

# 幻想崩壞（Broken Phantasm）

將龐大的魔力當作火藥點燃，把整個寶具當作炸彈引爆來攻擊。

對一般的Servant有號稱一擊必殺的破壞力，但一經使用寶具就會損毀。

只能以實體寶具發動。將該寶具破壞，直接造成魔術傷害。

１～３級寶具：造成【10d6+等級×5】點傷害。

４～６級寶具：造成【20d6+等級×5】點傷害。

７～９級寶具：造成【30d6+等級×5】點傷害。

引爆的傷害範圍為15×15。發動的範圍沒有限制。可以將之擲出、射出後才引爆。

發動時必須支付【寶具等級×10MP】的消耗。

被發動幻想崩壞的寶具在特異點結束前無法修復。之後，該角色下一個主線章節開始時自動修復，但ＭＰ將以最大值的一半開始。

# 孤注一擲

選擇迴避的場合：傷害後的HP會固定為1點。

各等級幸運之發動率、次數上限如下。

    Ｅ／１０％‧１次。

    Ｄ／２０％‧１次。

    Ｃ／３０％‧２次。

    Ｂ／４０％‧２次。

    Ａ／５０％‧３次。

    ＥＸ／８０％‧５次。

# 大成功與大失敗

## 大成功

    當任意的d6骰有半數以上的骰值為5或6，將根據符合該條件的骰子數量，再追加擲骰。擲骰完畢後，將原始值骰加上額外骰值。

    12d6以內的判定骰，必須要有至少6個骰子達成條件。

    追加擲骰時，若是再度觸發，同樣適用追加。【大成功】追加的擲骰不會觸發【大失敗】。

    Ｃ級的幸運將減少１個骰子的觸發門檻，Ａ級則減少２個（最低依舊為６個，追加擲骰不適用）。

    多段傷害（包含游擊、次數系攻擊元素）不會觸發大成功。反擊藉由大成功得到的額外骰數僅會用於抵消傷害，而無法使對方受到傷害。


## 大失敗

    當任意的d6骰有半數以上的骰值為1或2，將根據符合該條件的骰子數量，再追加擲骰。擲骰完畢後，將原始骰值減去額外骰值。

    12d6以內的判定骰，必須要有至少6個骰子達成條件。

    追加擲骰時，改將5、6視為觸發條件，若是再度觸發，同樣適用追加。【大失敗】追加的擲骰不會觸發【大成功】。

    Ｃ級的幸運將增加１個骰子的觸發門檻，Ａ級則增加２個（最高等同於骰數，追加擲骰不適用）。

    多段傷害（包含游擊、次數系攻擊元素）不會觸發大失敗。

骰子的最大骰數不足６顆的場合，全數骰值皆為1、2的場合必然視為大失敗；全數骰值皆為5、6的場合必然視為大成功。在d4的場合也適用，骰值要求也做出相應變動。

【大成功】的計算優先於【大失敗】。

# 破損度

一般武器、灰鎖、黑鍵、魔術禮裝、教會武裝、實體寶具或依靠實體物施放的寶具在受到攻擊時會累積破損度。

承受上限依照等級（１～９）依序為：

    １級～２級／１００

    ３級／２００

    ４級／３００

    ５級／４００

    ６級／５００

    ７級／６５０

    ８級／８００

    ９級／１０００

僅部分行動會進行累積破損的判定：不反應、[防禦](#防禦)、[反擊（雙方）](#反擊)、[阻擋](#阻擋)、[迴避（成功的場合只累積一半）](#迴避)、[保護](#保護)。

不過，進行反擊的場合，只有當下用於反擊的武器會累積破損度。

透過元素或職階特性造成的額外破損，一次行動只作用一次。

骰值受到影響的場合，所能造成的破損度不會超出【原始骰數＊６】。

寶具不會被常規手段造成損傷。但是，若對手以同等級（含）以上的寶具進行攻擊，就會根據原始傷害累積破損度。被實體寶具攻擊的場合，一般攻擊或戰技也能夠累積破損度。

依照破損度的累積程度會有不同影響：

    ０％～５０％：還能正常使用。

    ５１～９９％：雖然能正常使用，但是會失去額度消耗最多的其中一個效果（傷害以外）。

    １００％：完全破壞，無法使用。後續使用必須先修復到０％。

可以在[休息](#基礎休息行動)時進行修復，每１ＭＰ可以消去１點破損度。

# 對空單位的相關判定

若是可以飛行的單位，移動時可以任意宣言是否變動自身的高度。

飛行中的單位每被一人抓住，飛行高度減少５Ｍ，移動格數－１（最低至０）。

飛行的一方無法迴避擒抱者的近戰攻擊。

    起跳：消耗1AP。以筋力對應跳躍距離為：0.5M以下／1M以下／1M／2M／3M／10M。

    從者起跳：消耗1AP。以筋力對應跳躍距離為：2M／5M／10M／20M／30M／60M。

    擒抱：接觸到的時候可以進行擒抱。有【自身敏捷＊２０％－目標敏捷＊１０％】的成功率。成功時視為與對方處同一高度。

    遭到擒抱的對象每回合可以有一次機會，支付０～３ＡＰ嘗試甩下對方，有【自身筋力×１０％＋消耗ＡＰ×１０％－對方筋力×１０％】的機率成功。

    弓箭射擊：具有射程限制。以筋力對應射擊距離為：5M／10M／15M／20M／25M／50M。從者的射擊距離為四倍。

    槍械射擊：具有射程限制。以槍械等級對應射擊距離為：20M／30M／40M／50M／60M／70M／80M（３～９級）。

在空中遭到遠程攻擊時的防禦行動效益減半。

# 日常消耗

在章節進行時，若ＧＭ主觀判定從者已經維持了過久的實體化，休息前將會扣減１０～４０不等的ＭＰ，視情況可能再做提升。

# 過度消耗

一旦從者在短時間內過度抽取御主的魔力，將可能使御主陷入生命危險。

從者一次性消耗御主３０點（含）以上的ＭＰ時，將使其損失１０點ＨＰ（可疊加）。

御主的ＭＰ已經歸零的場合，若仍然被抽取魔力，將以ＨＰ代替ＭＰ進行消耗。

# 御主的成長與改變

當御主完成了一個章節，ＧＭ將會審視在該章節中的扮演情形，並對於副素質做出調整。

若是一個普通人御主在章節中聽聞了許多魔術世界的知識，就有可能提升智慧的等級。

但若是魔術師或教會勢力的御主對於有其必要性的戰鬥、殺傷行為感到抗拒，就有可能降低覺悟或是冷酷的等級。

# 大型單位規則

對於因為適用某些效果而改變為1×1以外大小的單位，將同時得到以下影響。

    - 其單位每增加一圈大小，降低其１點ＡＰ上限（最低為５）。
    - 其單位每增加一圈大小，降低其１格移動上限（最低為１）。
    - 其單位每增加一圈大小，受到的範圍傷害上升５０％。

# 選用規則：行動難度

是否以此作為評斷標準以ＧＭ決定為準。

部分需要依靠素質的劇情行動，將按照等級提供難度，並擲1d20。各個情況有不同的難度門檻，需要大於該門檻才能視為成功。

Ｅ級～Ｄ級：＋１／Ｃ級～Ｂ級：＋２／Ａ級：＋３／ＥＸ級：＋５。

由從者引發的敵性判定，御主的行動難度＋２。

素質帶有減號的場合，固定以原等級對應值－１計算。

素質帶有加號的場合，每個加號額外增加一次擲骰機會，在那些結果中依照「大成功＞大失敗＞成功＞失敗」的優先度決定使用的結果。

默認骰到20視為大成功，骰到1視為大失敗，是否具有額外效果，由ＧＭ根據當下情況判斷。

此外，只在進行幸運的d20骰時，將由GM進行暗骰。對此發動重骰的場合，一個重骰額度會額外骰出２個結果。

在宣言重骰前不會告知是否成功，但重骰時會依照「大成功＞大失敗＞成功＞失敗」的優先度決定使用的結果。

以百面骰進行的場合，成功機率默認為【對應素質等級×１０％】，且每個＋號視為增加５％，每個－號視為減少５％。
